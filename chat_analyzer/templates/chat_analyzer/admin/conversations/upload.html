How to Export Whatsapp Chat
  1. Open the WhatsApp chat you want to export
  2. Tap the menu (three dots) or group name
  3. Select "More" -> "Export Chat"
  4. Choose "Without Media"
  5. Save the .txt file to your device
  6. Upload it using the form above
Tips for Best Results
  • Messages are automatically linked to clients by phone number or username
  • Make sure client phone numbers are correctly entered in their profile
  • For saved contacts, the contact name must match the client's username
  • Supported format: [13/10/2024, 23:23:42] Sender: Message
Example Format
[13/10/2024, 09:30:15] +60 16-935 4580: Hello, I need help
[13/10/2024, 09:32:22] Admin: Hi! How can I help?
[13/10/2024, 09:35:45] +60 16-935 4580: My child needs therapy
Both bracket format and dash format are supported
